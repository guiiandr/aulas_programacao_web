Questão 1:

O que é uma regra CSS? Explique o conceito de regra CSS e forneça um exemplo prático que inclua um seletor, uma propriedade e um valor.

Uma regra CSS é uma instrução que define como os elementos HTML devem ser apresentados em uma página da web.
Ela consiste em um seletor, que identifica quais elementos serão afetados, e um conjunto de declarações, que incluem propriedades e valores que especificam o estilo desejado.

Exemplo:
h1 { color: blue; font-size: 24px; }

Seletor: h1 (aplica-se a todos os elementos H1 da página).
Propriedades: color e font-size.
Valores: blue e 24px.

Questão 2:

O que são seletores CSS? Quais são os tipos existentes?.

Seletores de Tipo: Selecionam todos os elementos de um tipo específico (ex: p, h1, div).
Seletores de Classe: Selecionam elementos com uma classe específica (ex: .minha-classe).
Seletores de ID: Selecionam um único elemento com um ID específico (ex: #meu-id).
Seletores de Descendente: Selecionam elementos que são descendentes de um elemento pai (ex: div p).
Seletores de Filho: Selecionam filhos diretos de um elemento pai (ex: ul > li).
Seletores de Irmão Adjacent: Selecionam um elemento que é irmão imediato de outro (ex: h1 + p).
